Gizmo is an effortless way to develop Web applications and RESTful servies.

Best of Clojure

Under the hood, Gizmo relies on the libraries that have been running in production by many people from Clojure community: Ring, Compojure, Enlive, route_one and others.

Fast

With parallel widget rendering, independent parts of the page will be prepared in parallel without any effort from your side.

Battaries included

Gizmo provides you with a great toolchain to quickstart application development, providing a ways to simplify routing, templating, serialization, built-in services and more.

View-aware

Gizmo will find all missing HTML templates and parts of the page that application code relies on, that were removed from HTML markup, so you can always be certain that your code does what it should.

Sane conventions

Gizmo provides you with a set of simple conventions that help you to keep application concerns separated, making it extremely easy to find parts you need to work on.

Serivce-driven

Split long-running parts of your application to lightweight cooperative services, controllable via minimalistic API.
